# YouTube Transcript Overlay

YouTube の動画ページで、文字起こしを動画そのものの上に字幕風に重ねて表示する Chrome 拡張です。

## できること

- 動画上に字幕をオーバーレイ表示
- シークバー右端の少し上にあるコピーアイコンから、元字幕全文を 1 文としてコピー
- 再生位置に合わせて字幕を切り替え
- 文字起こしを 5 ブロックずつ固定で連結して表示
- Gemini / Z.AI を使って 3 ブロック単位で順番に翻訳
- メイン字幕が `日本語（自動生成）` の場合はオーバーレイを自動で非表示
- シーク時は未翻訳の現在位置ブロックを優先して翻訳再開
- 字幕をドラッグして移動
- ダブルクリックで位置を初期化
- 文字サイズ、背景の濃さ、字幕幅を調整
- 翻訳の有効化、プロバイダ、API Key、モデル名を歯車パネルから設定
- ステータスは通常アイコン表示、必要時だけ展開

## インストール

1. このフォルダを任意の場所に置く
2. Chrome で `chrome://extensions/` を開く
3. 右上のデベロッパーモードをオン
4. `パッケージ化されていない拡張機能を読み込む` を押す
5. このフォルダを選ぶ

## 使い方

- YouTube の動画ページを開くと自動で文字起こしを読み込みます
- 字幕をそのままドラッグすると位置を移動できます
- ダブルクリックで位置を初期化できます
- 右上の歯車から見た目を調整できます
- シークバー右端の少し上にある `⎘` アイコンを押すと、オリジナル言語の字幕を全ブロック連結したテキストをコピーできます
- 翻訳を使う場合は歯車から `翻訳を有効化` をオンにして、`Gemini` または `Z.AI` を選び、対応する API Key とモデル名を入力してください
- Z.AI の coding plan を使う場合は専用 endpoint `https://api.z.ai/api/coding/paas/v4` へ接続します
- 翻訳がまだ終わっていない区間は、翻訳完了まで原文が表示されます
- メイン字幕が `日本語（自動生成）` と判定された動画では、YouTube 側の字幕オン/オフに関係なくこの拡張のオーバーレイは表示されません
- ステータスアイコンにカーソルを合わせると詳細メッセージを確認できます

## 補足

- 連結は固定の 5 ブロック単位です
- 翻訳 API への送信は固定の 3 ブロック単位です
- 翻訳は逐次実行です。並列実行はしません
- Gemini は structured output を使って JSON 配列で受け取ります
- Z.AI は chat completions 互換 endpoint に JSON 配列のみを返すよう指示して受け取ります
- 1〜5 の次は 6〜10 になり、2〜6 のようなローリング表示はしません
- YouTube 側の DOM が変わると修正が必要になる場合があります
- API Key は `chrome.storage.local` に保存されます
