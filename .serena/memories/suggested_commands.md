# Suggested commands
- `ls -la` : inspect repository contents.
- `rg --files` : list tracked files quickly.
- `node --check content.js` : syntax check the main content script.
- `node --check background.js` : syntax check the background service worker.
- `node --check page-bridge.js` : syntax check the bridge script.
- Open `chrome://extensions/` in Chrome and load this folder as an unpacked extension for manual verification.
